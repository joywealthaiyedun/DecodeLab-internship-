# DecodeLab-internship-
Intern
## Overview

This repository contains three end-to-end decodelab data analytics internship projects completed using Excel and SQL.

The projects demonstrate the complete analytics workflow from data cleaning and preparation to exploratory data analysis and SQL-based business intelligence.

Dataset: E-Commerce Sales Dataset (1,200 Orders)

---

# Project 1: Data Cleaning & Preparation (Excel)

## Objective

Prepare raw data for analysis by identifying and resolving data quality issues.

## Tasks Performed

* Checked for missing values
* Identified duplicate records
* Verified unique Order IDs
* Validated data types
* Checked calculation accuracy
* Checked  data quality

## Findings

* 309 missing values found in CouponCode
* No duplicate records detected
* No duplicate Order IDs
* No invalid values identified
* Dataset was analysis-ready

## Skills Demonstrated

* Data Cleaning
* Data Validation
* Excel Functions
* Data Preparation

# Project 2: Exploratory Data Analysis (Excel)

## Objective

Explore the dataset to uncover patterns, trends, and business opportunities.

## Analysis Performed

* Descriptive Statistics
* Revenue Analysis
* Product Performance Analysis
* Customer Purchase Behavior
* Coupon Effectiveness
* Referral Source Analysis
* Trend Analysis

## Key Findings

### Revenue Performance

* Total Revenue: $1,264,761.96
* Average Order Value: $1,053.97
* Median Order Value: $879.76

### Product Analysis

Top Revenue Generating Products:

1. Chair
2. Printer
3. Laptop

### Marketing Insights

* Instagram generated the most orders
* Free Shipping was the most effective promotion

### Customer Behavior

* Average cart size: 5.4 items
* Coupon usage rate: 74.25%

### Operational Insights

* Cancellation and return rate reached 41.4%

## Skills Demonstrated

* Pivot Tables
